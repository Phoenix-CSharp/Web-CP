О проекте

Данный проект разработан в рамках учебной дисциплины "WEB-разработка" студентом 2 курса ИШИТР ИСиТ ТПУ Лилик Глебом Егоровичем.

Цели проекта

  • Демонстрация навыков адаптивной верстки с использованием Bootstrap 3
  • Применение современных технологий фронтенд-разработки
  • Создание интерактивных элементов пользовательского интерфейса
  • Организация многостраничного веб-сайта
  • Использование Node.js для запуска локального сервера

Технологии

HTML5

Семантическая верстка, современные элементы

CSS3

Flexbox, Grid, анимации, медиа-запросы

Bootstrap

Адаптивная сетка, компоненты

JavaScript

Интерактивность, обработка событий

Node.js

Локальный сервер, обработка запросов

NPM

Управление зависимостями
